# Three Tribes Chronicle

A text-driven RPG with RPG progression, simulation elements, and a mystical tribal world.

## Getting Started

### Installation

```bash
npm install
```

### Build

```bash
npm run build
```

### Run

```bash
npm start
```

### Development

```bash
npm run dev
```

## Game Overview

You begin as a civilian within one of three tribes and develop through skill trees, jobs, and choices that influence the tribe's fate.

Each in-game day, you spend stamina on actions — farming, hunting, gathering, trading, exploring, or visiting NPCs — to progress your character and influence your tribe's overall development.

## Version 0.1 Features

- Stonefang tribe
- Civilian job
- 4 actions: Farm, Gather, Trade, Visit NPC
- 3 NPC types
- Basic random event system
- Save/load functionality
